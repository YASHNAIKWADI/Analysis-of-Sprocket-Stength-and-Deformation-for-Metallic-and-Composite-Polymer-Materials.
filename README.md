# Analysis-of-Sprocket-Stength-and-Deformation-for-Metallic-and-Composite-Polymer-Materials.
Comparative FEA of a chain sprocket using C45 carbon steel, PA66-GF30, and Kevlar-49 performed in ANSYS. Deformation, von Mises stress, mass, and strength-to-weight ratio are evaluated under identical loading conditions.
